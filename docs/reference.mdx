Complete reference for the `@kiyasov/elysia-nest` package and all its subpath modules.

## Core

<CardGroup cols={1}>
  <Card title="Controllers" icon="server" href="/core-concepts/controllers">
    `@Controller`, `@Get`, `@Post`, `@Put`, `@Patch`, `@Delete`, `@All`, `@Head`, `@Options` — define HTTP routes with decorators.
  </Card>
  <Card title="Modules" icon="cubes" href="/core-concepts/modules">
    `@Module`, `@Global` — encapsulate controllers, providers, and imports into cohesive units.
  </Card>
  <Card title="Dependency Injection" icon="syringe" href="/features/dependency-injection">
    `@Injectable`, `@Inject`, `@Optional` — constructor-based DI with singleton, transient, and request scopes.
  </Card>
  <Card title="Parameter Decorators" icon="brackets-curly" href="/features/parameter-decorators">
    `@Body`, `@Param`, `@Query`, `@Headers`, `@Req`, `@Res`, `@Ip`, `@HostParam` — extract data from incoming requests.
  </Card>
  <Card title="HTTP Decorators" icon="arrow-right-arrow-left" href="/features/http-decorators">
    `@HttpCode`, `@Header`, `@Redirect`, `@Schema` — control response metadata and validation.
  </Card>
  <Card title="Guards" icon="shield-halved" href="/features/guards">
    `@UseGuards`, `CanActivate` — protect routes with authorization logic.
  </Card>
  <Card title="Interceptors" icon="arrows-split-up-and-left" href="/features/interceptors">
    `@UseInterceptors`, `ElysiaInterceptor` — transform requests and responses in the pipeline.
  </Card>
  <Card title="Pipes" icon="pipe" href="/features/pipes">
    `@UsePipes`, `PipeTransform` — validate and transform incoming data.
  </Card>
  <Card title="Middleware" icon="layer-group" href="/features/middleware">
    `@Middleware`, `ElysiaNestMiddleware` — class-based middleware with full `next()` chain support.
  </Card>
  <Card title="Exception Handling" icon="triangle-exclamation" href="/features/exception-handling">
    `@Catch`, `ExceptionFilter`, `HttpException` and built-in HTTP exceptions.
  </Card>
  <Card title="Lifecycle Hooks" icon="rotate" href="/features/lifecycle-hooks">
    `OnModuleInit`, `OnApplicationBootstrap`, `OnModuleDestroy`, `BeforeApplicationShutdown`.
  </Card>
</CardGroup>
